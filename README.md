## Шаг 2. Установка зависимостей
1) Перейдите в папку проекта
2) Установите зависимости. Для этого:
2.1) Откройте терминал в папке проекта
2.2) Введите команду "pip install virtualenv"
2.3) Введите команду "virtualenv venv"
2.4) Установите библиотеки, для этого введите команду "pip install -r requirements.txt"

## Шаг 3. Запустите скрипт для скачивания статей
1) Запустите файл main.py

В ходе работы приложения будут выполнены следующие действия:
* программа начнет поиск 100 статей на сайте Habr. Она найдет только существующие, не удаленные статьи
* для каждой из найденных ссылок на статью будет осуществлена запись строки в файл "index.txt" в следующем формате: "{порядковый номер статьи},page_{порядковый номер статьи}.txt,{ссылка на веб-страницу}"
например, "1,page_1.txt,https://habr.com/ru/articles/312884/"
таким образом, файл index.txt содержит информацию о порядковом номере статьи, имени файла с текстом статьи (который будет выгружен далее) и ссылкой на веб-страницу, содержащую статью
* каждая веб-страница будет полностью скачана в папку "pages"

## Шаг 4. Запустите скрипт для выделения текста
1) Запустите файл remove_from_pages.py

В ходе работы приложения будут выполнены следующие действия:
* Программа удалит все теги из HTML-разметки страницы, содержащие ссылки на изображения, файлы стилей, файлы скриптов, мета-информацию
* Очищенные статьи сохранятся в папку "pages_removed"
