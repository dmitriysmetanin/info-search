article-formatted-body_version-1"><div xmlns="http://www.w3.org/1999/xhtml">Поиск поддоменов — неотъемлемая часть подготовки ко взлому, а благодаря некоторым инструментам противостояние этим действиям стало намного проще.<br/>
<br/>
Незащищенные поддомены подвергают вашу деятельность серьезной опасности, а в последнее время произошел целый ряд инцидентов, при которых взломщики воспользовались поддоменами для обхода защиты.<br/>
<br/>
В случае последнего из череды инцидентов весь код сайта <a href="https://avicoder.me/2016/07/22/Twitter-Vine-Source-code-dump/">Vine</a> можно было загрузить с <a href="https://geekflare.com/schedule-website-security-vulnerabilities-scan/">незащищенного</a> поддомена.<br/>
<br/>
Если вы владелец сайта или изучаете вопросы информационной безопасности, вы можете воспользоваться следующими инструментами чтобы найти поддомены любого домена.<br/>
<br/>
<br/>
<a name="habracut"></a><br/>
<i>От переводчика:<br/>
Надеюсь, что данные инструменты помогут Вам сохранить информацию на ваших <a href="https://vds.menu/">виртуальных</a> и <a href="https://dedicated.menu/">выделенных</a> серверах.</i><br/>
<br/>
Список инструментов:<br/>
<br/>
<ol>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#Censys">Censys</a></li>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#Pentest-Tools">Pentest-Tools</a></li>

<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#CloudPiercer">CloudPiercer</a></li>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#Detectify">Detectify</a></li>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#SubBrute">SubBrute</a></li>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#Knock">Knock</a></li>
<li><a href="https://habrahabr.ru/company/hosting-cafe/blog/312954/#DNSRecon">DNSRecon on Kali Linux</a></li>
</ol><br/>
<a name="Censys"></a><b>1. Censys</b><br/>
В первую очередь для поиска поддоменов, как правило, прибегают к <a href="https://censys.io/">Censys</a>. Помимо поддомена вы можете найти еще много интересного, а именно:<br/>
<br/>
<ul>
<li>данные об IP (может быть полезно для <a href="https://geekflare.com/test-origin-ip-exposed/">поиска реального IP-адреса</a>);</li>
<li>данные о сертификате;</li>
<li>открытые порты;</li>
<li>SSL/TLS протокол рукопожатия и алгоритмы шифрования (полезно при поиске слабых шифров/протоколов).</li>
</ul><br/>
<br/>
<br/>
Вы получите неплохой обзор сведений о домене.<br/>
<br/>
<a name="Pentest-Tools"></a><b>2. Pentest-Tools</b><br/>
С помощью <a href="https://pentest-tools.com/information-gathering/find-subdomains-of-domain">Pentest-Tools</a> при поиске поддомена можно воспользоваться несколькими методами, например: передача зоны DNS, перебор DNS на основе списка слов или использование поисковой системы.<br/>
<br/>
<br/>
Результаты поиска можно сохранить в формате PDF.<br/>
<br/>
<a name="DNSDumpster"></a><b>3. DNS Dumpster</b><br/>
<a href="https://dnsdumpster.com/">DNS Dumpster</a> — это инструмент для поиска информации о домене и хосте. Авторы проекта — HackerTarget.com.<br/>
<br/>
Вы сможете найти сведения не только о поддомене, но и о DNS сервере, MX и TXT записях, а также получить графическое представление информации о вашем домене.<br/>
<br/>
<br/>
<br/>
<a name="Sublist3r"></a><b>4. Sublist3r</b><br/>
<a href="https://github.com/aboul3la/Sublist3r">Sublist3r</a> — это инструмент на языке Python для обнаружения поддоменов с помощью поисковых систем. На сегодняшний день Sublist3r поддерживает Google, Yahoo, Bing, Baidu, Ask, Netcraft, Virustotal, ThreatCrowd, DNSdumpster and PassiveDNS.<br/>
<br/>
Sublist3r поддерживает только Python версии 2.7 и зависит от нескольких библиотек.<br/>
<br/>
Вы можете использовать этот инструмент в Windows, CentOS, RedHat, Ubuntu, Debian и любой другой ОС на базе UNIX. Ниже приводится пример для CentOS.<br/>
<br/>
<ul>
<li>Зайдите на ваш Linux-сервер;</li>
<li>Скачайте последнюю версию <a href="https://github.com/aboul3la/Sublist3r/archive/master.zip">Sublist3r</a></li>
</ul><br/>
<pre><code class="bash">wget https://github.com/aboul3la/Sublist3r/archive/master.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Распакуйте скачанный файл:<br/>
<br/>
<pre><code class="bash">unzip master.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<ul>
<li>Будет создана новая папка “Sublist3r-master”</li>
</ul><br/>
Как я ранее упоминал, существуют следующие зависимости, для установки которых можно воспользоваться командой yum:<br/>
<br/>
<pre><code class="bash">yum install python-requests python-argparse
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Теперь все готово для обнаружения поддоменов с помощью следующей команды:<br/>
<br/>
<pre><code class="bash">./sublist3r.py -d yourdomain.com
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<br/>
<br/>
Как видите, инструмент обнаружил мои поддомены.<br/>
<br/>
<a name="Netcraft"></a><b>5. Netcraft</b><br/>
<a href="http://searchdns.netcraft.com/">Netcraft</a> располагает обширной базой данных о доменах и ее не стоит обходить стороной при поиске открытой информации о поддоменах.<br/>
<br/>
<br/>
<br/>
Результат поиска будет содержать всю информацию о домене и поддоменах, в том числе дату первого просмотра, диапазоне адресов и информацию об операционной системе. Если вам необходимо получить больше информации о сайте, просто откройте отчет о сайте и вам будет предоставлена уйма информации о технологиях, рейтинге и т.д.<br/>
<br/>
<br/>
<br/>
<a name="CloudPiercer"></a><b>6. CloudPiercer</b><br/>
<a href="https://cloudpiercer.org/">CloudPiercer</a> может иногда оказаться полезным при поиске информации о том, существует ли поддомен вашего домена. Кстати, CloudPiercer — это потрясающий и простой способ узнать защищен ли фактический IP-адрес вашего сайта.Открытая информация об IP-адресе делает ваш сайт уязвимым для DDoS-атак.<br/>
<br/>
<a name="Detectify"></a><b>7. Detectify</b><br/>
<a href="https://detectify.com/">Detectify</a> осуществляет поиск поддоменов по предопределенному списку из нескольких сотен слов, но только в случае, если вы являетесь собственником домена. Тем не менее, если вы являетесь зарегистрированным пользователем Detectify, вы сможете включить функцию обнаружения поддоменов в разделе overview в настройках.<br/>
<br/>
<div style="text-align:center;"></div><br/>
<a name="SubBrute"></a><b>8. SubBrute</b><br/>
<a href="https://github.com/TheRook/subbrute">SubBrute</a> — это один из самых популярных и точных инструментов перечисления поддоменов. Проект разработан сообществом и использует открытый определитель имен в качестве прокси, так что SubBrute не отправляет трафик на целевой DNS-сервер.<br/>
<br/>
Это не онлайн-инструмент, так что вам придется установить его на ком��ьютер. SubBrute можно использовать на Windows или UNIX системах. Программу установить очень легко. Ниже пример для CentOS/Linux.<br/>
<br/>
<ul>
<li>Зайдите на ваш Linux-сервер</li>
<li>Скачайте последнюю версию <a href="https://github.com/TheRook/subbrute/archive/master.zip">SubBrute</a></li>
</ul><br/>
<pre><code class="bash">wget https://github.com/TheRook/subbrute/archive/master.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<ul>
<li>Распакуйте скачанный zip-файл</li>
</ul><br/>
<pre><code class="bash">unzip master.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Будет создана новая папка “subbrute-master”. Зайдите в папку и выполните subbrute.py с необходимым доменом.<br/>
<br/>
<pre><code class="bash">./subbrute.py yourdomain.com
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Операция займет несколько секунд и отобразятся найденные поддомены.<br/>
<br/>
<a name="Knock"></a><b>9. Knock</b><br/>
<a href="https://github.com/guelfoweb/knock">Knock</a> — еще один инструмент на языке Python для обнаружения поддоменов. Он протестирован для Python 2.7.6. Knock находит поддомены целевого домена по списку слов.<br/>
<br/>
<ul>
<li>Knock можно установить на ОС на базе Linux.</li>
</ul><br/>
<pre><code class="bash">wget https://github.com/guelfoweb/knock/archive/knock3.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<ul>
<li>Распакуйте скачанный zip-файл командой unzip</li>
</ul><br/>
<pre><code class="bash">unzip knock3.zip
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<ul>
<li>В результате будет создана новая папка “knock-knock3”</li>
<li>Зайдите в папку и установите с помощью следующей команды</li>
</ul><br/>
<pre><code class="bash">python setup.py install
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
После установки вы можете производить поиск поддоменов следующим образом:<br/>
<br/>
<pre><code class="bash">./knockpy.py yourdomain.com
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
<a name="DNSRecon"></a><b>10. DNSRecon для Kali Linux</b><br/>
<a href="https://geekflare.com/kali-linux-installation-guide-vmware/">Kali Linux</a> — это отличная платформа для оценки информационной безопасности и на ней можно использовать DNSRecon без дополнительной установки каких-либо инструментов.<br/>
<br/>
DNSRecon проверяет все NS-записи на предмет смены зон, общие записи DNS, обработку шаблонов, PTR-записи и т.д.<br/>
<br/>
Чтобы воспользоваться DNSRecon, просто выполните следующую команду<br/>
<br/>
<pre><code class="bash">dnsrecon –d yourdomain.com
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
