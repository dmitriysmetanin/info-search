article-formatted-body_version-1"><div xmlns="http://www.w3.org/1999/xhtml">Еще недели не прошло с анонса новых методов api вконтакте. Для владельцев сообществ выкатили возможность подписывать пользователей на уведомления от сообщества, что дает право владельцам писать подписчикам неограниченно. Не за горами приложения, ака рассылки по подписчикам сообщества, со статистикой, шаблонами, отложенной рассылкой и вообще. <br/>
<br/>
Бренды, блогеры и еще кто-нибудь начнут собирать базы подписчиков. В этой статье я расскажу как создать крохотное, но функциональное приложение генерирующее лендинг c кнопкой подписки на сообщения от сообщества. Ака лендинг пэйдж для посетителей с вк.<br/>
<br/>
<div style="text-align:center;"></div><br/>
<a name="habracut"></a><br/>
<h2>Погнали</h2><br/>
Нам потребуется минут 15 на сборку, домен, хостинг, токен-ключ сообщества вк и сколько-то времени для CSS-стилей.<br/>
<br/>
Токен-ключ нужен для управления сообществом и отправки запросов к вк api от имени сообщества. Нам он потребуется один раз, чтобы узнать id сообщества пользователя. Создать его просто, нужно зайти в управление сообществом вконтакте и нажать создать ключ в разделе работа с api. Сделайте это в одном из своих сообществ или создайте новое под это дело. <br/>
<br/>
<div style="text-align:center;"></div><br/>
<h2>Полученный токен не для чужих глаз</h2><br/>
Код виджета «Разрешить писать сообществу» очень прост и выглядит следующим образом. <br/>
Где 53495256 — это уникальный id сообщества. Вместо этих цифр мы подставим id любого сообщества для которого генерируем страницу подписки.<br/>
<br/>
<pre><code class="javascript">&lt;script type="text/javascript" src="//vk.com/js/api/openapi.js?133"&gt;&lt;/script&gt;
&lt;!-- VK Widget --&gt;
&lt;div id="vk_allow_messages_from_community"&gt;&lt;/div&gt;
&lt;script type="text/javascript"&gt;
VK.Widgets.AllowMessagesFromCommunity("vk_allow_messages_from_community", {height: 30}, 53495256);
&lt;/script&gt;
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Чтобы не писать для пользователей приложения огромных инструкций как получить id их сообщества мы сделаем все за них. Проще попросить их скопировать короткий адрес сообщества и создать ссылку вида домен_вашего_приложения/короткий_адрес/, где короткий адрес — это адрес их сообщества вконтакте. <br/>
<br/>
Если кто-то будет заходить напрямую на ваш сайт, то будет подгружаться то сообщество, чей токен вы используете. И обязательно в настройках каждого сообщества, кто будет пользоваться вашим приложением, должна быть включена опция разрешить использование виджета.<br/>
<br/>
<div style="text-align:center;"></div><br/>
Нам остается немного:<br/>
<br/>
<ol>
<li>Получить короткий адрес сообщества из строки браузера.</li>
<li>Сделать запрос к api вк и узнать id сообщества.</li>
<li>Подставить полученные данные в виджет.</li>
</ol><br/>
Создадим файл .htaccess и зададим конфигурацию, чтобы сервер понимал наш замысел.<br/>
<br/>
<pre><code class="apache">&lt;IfModule mod_rewrite.c&gt;
RewriteEngine on
RewriteBase /
RewriteRule ^([^/]+)/$ /index.php?vkgr=$1 [L]
&lt;/IfModule&gt;
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
Тогда для нас ссылка домен_вашего_приложения/habr/ превратится в домен_вашего_приложения/?vkgr=habr. Получим данные из GET. Теперь соберем запрос через file_get_contents() к вк api и узнаем некоторые данные о сообществе. Возьмем название, описание и количество пользователей. Так на странице мы сможем добавить больше информации для посетителей, чем просто кнопку виджета «Разрешить писать сообществу». ВКонтакте возвращает данные формате json, нужно данные обработать.<br/>
<br/>
<pre><code class="php">if (isset($_GET['vkgr'])) {
	$group = htmlspecialchars(trim($_GET['vkgr']));
}
// токен вымышленный, вставьте свой
$token = "b98908abcbcbasbbxabbcbabbabbb247823479823aasdasdwe345345345345354353345345345345353453";
$fields = "description,members_count";
$v = 5.58;
$url = 'https://api.vk.com/method/groups.getById?group_id='.$group.'&amp;fields='.$fields.'&amp;access_token='.$token.'&amp;v='.$v;
$about = file_get_contents($url);
$result = json_decode($about, true);
</code><div class="code-explainer"><a href="https://sourcecraft.dev/" class="tm-button code-explainer__link" style="visibility: hidden;"></a></div></pre><br/>
